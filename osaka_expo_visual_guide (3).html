🎬 万博における映像技術の進化系譜

モントリオール万博→大阪万博→筑波万博→愛知万博まで

🌍 モントリオール万博（1967年）
📊 基本情報
開催時期： 大阪万博の3年前
来場者数： 約5000万人
評価： 万博史上での大成功
🎪 会場規模と設備
特徴： 相当広大な敷地での開催
交通網： モノレール的な鉄道システムで会場を輸送でネットワーク
効果： 大規模敷地を有機的に繋ぐインフラストラクチャー
🎬 注目映像作品
主要作品：
• In the Labyrinth（迷宮の中へ）
• The Place to Stand
• Polar Life

評価： 当時の映像マニアの間では知名度高い
ただし、一般認知は低い
🎯 「映像の歴史的挑戦」として記録されるべき作品群
🇯🇵 大阪万博（1970年）- 映像技術の成熟
🗼 太陽の塔と象徴性
設計者： 岡本太郎
当初計画： 「万博タワー」が象徴だった
存続の経緯： 一人の少年の手紙により、永久保存が決定

意味： 大屋根の「統一と均衡」を突き抜ける存在
→ 岡本太郎の前衛的思想を象徴
📹 マルチスクリーン技術の出現
技術革新： モントリオール万博から大阪万博で成熟
形態： 複数スクリーン（6面・8面など）での同期映像表現

その後のレガシー：
→ 博物館などで長年採用され続ける
→ 地方の築年数の長い施設に現存する例も多数
🎥 日本館の技術的挑戦
監督： 市川崑、安坂村（撮影者）
挑戦内容： マルチスクリーン撮影用に、総重量380kgのカメラ8台を車に搭載

意味： 現代人が容易に映像制作する背景には、こうした歴史的挑戦がある
📚 当時の記録： 通商産業省（現経産省）のレポートに詳細が記載
→見たことなくても「想像できる」素晴らしい文章として価値がある
🎬 過去の挑戦を知ることで、現在の映像技術の進化が理解できる
🔬 筑波万博（1985年）- 3D映像技術の誕生
🎭 来場者の世代変化
大阪万博の15年後。実際に訪問した来場者も出始める時代
🎮 3D眼鏡（アナグリフメガネ）の登場
発明者： ローマン・クローター（IMAXの設立創設者）
作品： 富士パビリオンでのスクリーン体験

社会的影響：
→ 90年代の子どもたちがアナグリフメガネを経験
→ 「3D映像を見るには特殊な眼鏡が必要」という認識の定着
🌐 ドームスクリーンの価値発見
問題： 平面スクリーンだと、死角が生じてしまう
解決： ドーム形スクリーンで全体を視野に収める
効果： より没入的で、360度的な視聴体験
🎯 「見えない部分をなくす」という単純だが革新的な発見
🌸 花の万博（1990年・大阪近郊）- 伝説的映像
🎬 富士パビリオンの伝説
特徴： 筑波万博に続き、富士パビリオンで伝説的映像作品を展示
意義： 万博が映像芸術の重要なプラットフォームであることを再証明
🤖 愛知万博（2005年）- メディアアート確立
📊 基本情報
来場者経験： 当時、複数回（3回以上）来場した人も存在
映像体験： 現場での体験と、後のYouTube視聴の並行
🎨 イマーシャルパノラマ技術
技術： 全周映像表現（360度ではなく、広角パノラマ）
特徴： あらゆる角度からあらゆる映像を体験可能
革新性： 当時として画期的な没入体験
🗿 フィジカルとデジタルの融合
彫像数： 134体の彫像をフィジカルに配置
映像連携： 映像と彫像が空間内で同期
効果： 「新しいメディアアート」としての成立
🎬 制作チーム
映像監督： 押井守（『ゴーストインザシェル』『パトレイバー2』『イノセンス』）
音楽： 河合賢治

→ 両者ともに映像・音響表現の第一人者
🔄 革新的な展示デザイン：「コンテンツの動的変化」
これまでの万博： 固定されたコンテンツを展示
愛知万博での工夫： 2ヶ月ごとに映像が変わる

意義：
→ 万博会期中に新しい体験が生まれる
→ リピート来場の動機づけ
→ 当時としては珍しい試み（現在は当たり前）
🎯 「固定から動的へ」：展示コンテンツそのものが進化する万博へ
📈 万博映像技術の進化系譜
timeline title 万博における映像技術の進化 1967 : モントリオール万博 : 映像作品の実験的チャレンジ 1970 : 大阪万博 : マルチスクリーン技術の成熟 : 複数カメラシステムの実装 1985 : 筑波万博 : 3D眼鏡（アナグリフ）登場 : ドームスクリーンの導入 1990 : 花の万博 : 伝説的映像作品の継続 2005 : 愛知万博 : メディアアート確立 : イマーシャルパノラマ技術 : コンテンツの動的変化 2025 : 大阪関西万博 : ゲザムトクンストベルク実現 : 生成AI × 総合芸術 : 世界初規模の統合表現
🎁 万博の本質的価値：「レガシーの継承」
❓ 「万博の成功/失敗」を語ることの意味
従来的評価： 来場者数や経済効果で判断
本質的評価： その後、社会にどう継承されるか
💫 レガシーとしての継続性
事例：マルチスクリーン技術
→ 大阪万博（1970年）で展示
→ 以降、数年〜10年単位で様々な施設に波及
→ 現在も地方の博物館に残存

事例：3D映像表現
→ 筑波万博（1985年）で導入
→ 90年代の子どもたちが経験
→ 映像文化の一部として定着
🌟 「5年後、10年後に『あれがあったね』と語り継ぐこと」がレガシー
💡 本質的な成功とは：
成功/失敗を語ること自体が愚か。
その代わり、どう継承されるか、語り継がれるかが重要。
それが次の世代の万博へと繋がる。
🚀 制約条件下での最適化：現代の万博制作
📏 制約の構造
パビリオンの存在： 建築空間がすでに確定している
フォーマットの固定： その制約内での最適化が必須

課題： 既存の「箱」を如何に活かすか
🎯 制約 = 創造性の源泉。制約の中にこそ、新しい表現が生まれる
📊 各万博の映像技術比較
万博 年 革新技術 後世への影響
モントリオール 1967 映像作品の実験的挑戦 映像マニアに知られる
大阪 1970 マルチスクリーン成熟 博物館等に長年採用
筑波 1985 3D眼鏡・ドームスクリーン 90年代子どもたちが経験
花の万博 1990 伝説的映像作品 映像アート文化の継続
愛知 2005 メディアアート + 動的コンテンツ 現代展示文化の確立
大阪関西 2025 ゲザムトクンストベルク + 生成AI 次世代映像文化の創造
📚 映像史の「知識継承」の重要性
🔍 なぜ過去を知る必要があるのか
現在の容易さ： 生成AIで簡単に映像が作れる時代

しかし実は：
→ 380kg のカメラ8台を搭載して撮影した時代
→ アナグリフメガネで3Dを体験した時代
→ ドームスクリーンに感動した時代

意味： そうした挑戦と工夫が、現在の「当たり前」を作った
🎬 現在の便利さの根底には、過去の多大な工夫がある
📖 推奨：通商産業省レポートの閲読
当時の制作背景や技術的工夫を記した歴史的文献として高い価値がある

🎬 この図解は、モントリオール万博（1967）から大阪関西万博（2025）に至る、

映像技術とメディアアート表現の進化系譜を解説したものです。

各万博で生まれた技術と表現が、社会に継承され、次世代へと繋がっていく。それがレガシーの本質。
