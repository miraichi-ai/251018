コンセプトデザインの3つの柱：構造・思想・文脈

🏛️ コンセプトデザインの必須要素
構造
どう体験させるか
空間、映像、音響、動線など、観客がどのような経路で、どのような順序で、何を体験するのかの全体設計
思想
何を問うのか
その作品を通じて、観客に何を考えさせるのか、どのような問いを投げかけるのか
文脈
なぜ今ここでなのか
歴史的背景、社会的状況、技術的発展など、この作品が今この瞬間に存在する必然性
🎯 この3つが揃ってはじめてコンセプトが成立する。どれか一つ欠けたら失敗する。
📋 3つの柱の具体的内容
🏗️ 構造（どう体験させるか）
実装レベルの設計：
• パビリオンの空間設計
• 映像の270度投影
• 音響のタイミング制御
• 観客の動線管理
• 時間的フロー構成

→ 「感覚的な経験」を物理的に設計する
💭 思想（何を問うのか）
哲学的レベルの設定：
• 人とAIの関係をどう再定義するのか
• 生命とは何か（命輝く未来社会）
• 技術と人間性の関係
• 個と全の相互関係

→ 観客に「考えさせる」ための問い設定
📚 文脈（なぜ今ここでなのか）
歴史的・社会的レベルの背景：
• 万博映像の55年の歴史（1967-2025）
• AI技術の急速な発展
• トランスヒューマニズムとポストヒューマニズムの台頭
• 「命輝く未来社会デザイン」というテーマの出現

→ 「なぜこの瞬間に必然なのか」を示す
🔗 統合的理解：
この3つは分離ではなく、相互に強化し合う関係にある
📊 100の情報を1つの形に：コンセプトデザインの圧縮力
🔍 万博という複雑な現象の見方
万博とは何か、という複数の視点：
• 国家発揚の場（政治的視点）
• 人間社会の在り方を問う場（人文学的視点）
• 祝祭（文化人類学的視点）
• 地球環境の未来を問う場（環境学的視点）
• 映像・技術の歴史（芸術・科学的視点）
• ヒューマニズムからトランスヒューマニズムへの移行期（哲学的視点）

→ これらすべてが「100の情報」に該当する
⚙️ 圧縮のプロセス
1. 複数の視点から情報収集
政治、文化、技術、哲学、歴史などの多領域から学ぶ

2. 要素の抽出と整理
100の情報から「本質的な要素」を抽出

3. 統合と表現
それらを「一つの感覚体験」に圧縮して表現

結果： 観客は複雑さを感じさせられずに、深い理解に到達
💎 複雑性を保ったまま圧縮する = 高度な知的作業
🧠 「知識を自分のものにする」という覚悟
⚠️ AIに頼ることの危険性
AIの利便性：
生成AIで素早く情報を取得できる

しかし：
AIが出した情報を「自分の言葉のように」話すのは危険
→ それは「理解」ではなく「引用」に過ぎない
📖 この講演における戦略
「Notion」や「Obsidian」などは一切使用しない

理由：
ナレッジデータベースに頼ると、「知識が外在化」してしまう

代わりに：
「頭の中に叩き込む」ことで、真に「自分のもの」にする

そうしてはじめて：
「なぜあの瞬間あれなのか」「これはこれなんです」と
裏打ちされた知識をさらっと説明できる
✅ AIの活用方法
適切な使い方：
• 素早く要点をつかむ（SSI利用）
• ファクトチェック（3～4回実施）
• 文献の確認（実際に読んで納得する）

その後：
得た情報を「完全に内在化」してからアウトプット
🎯 知識 = 自分の血肉化してこそ、初めて言語化できる
🌍 万博テーマ「命輝く未来社会デザイン」の哲学的読解
命輝く未来社会デザイン
トランスヒューマニズム × ポストヒューマニズム
🤔 テーマ解読の複層性
大阪関西万博の位置づけ：

「日本が元気なうちにできる最後の一般博」
↓
「AI時代の最初の万博」
↓
「人力でできる最後の万博」

→ これらは矛盾ではなく、同時に真実
ポストヒューマニズム（PostHumanism）
人間中心設計の解体

これまでの「人間が中心」という前提を問い直し、
人間とAI、人間と機械、人間と自然などの
関係を「平等な関係」として再構築する思想
🔄 映像制作への適用
従来的アプローチ：
「人間のための映像制作」「人間が主人公」

AIヒローズコレクション：
「人間とAIが共存する映像制作」
「どちらが主人公でもない共演」
💼 制作の実態：「まとまらないものの中で」
📝 理想と現実のギャップ
理想： 完璧に整理された映像
現実： 「正直全然まとまりがないもの」

しかし、それは「人間がなす技」だから当然
→ 完璧よりも「誠実さ」が重要
「100ぐらいの情報を受け取って、全部インプットして、その上で、全部把握した状態で言えるようにしなきゃダメだなっていうのがあって。『なぜあの瞬間あれなんですか？これはこれなんです』ってさらっと言えるみたいなそういうことが大事。」
🎯 言葉に詰まることの価値
許容される不完全さ：
言葉に詰まることはあってもいい

許容されない不完全さ：
知識が裏打ちされていない説明

→ 「思考の痕跡」が見える誠実な言語化が重要
❓ 最終的な問い系：「何を定義するのか」
🔑 映像を通じて「問う」こと
「何を問う」には複数のレベルがある：

1. 人とAIの関係をどのように再定義するか
2. 命とは何か、どう定義するか
3. 生命の価値をどう置き直すか
4. ポストヒューマニズムにおける人間の役割は何か

→ これらの「定義」が、作品の本質を決める
🎯 映像とは「問い」そのものであり、その問いが作品の芯になる

🏛️ この図解は、大阪関西万博AIヒローズコレクション制作における

「コンセプトデザイン」の本質的な方法論と哲学を解説したものです。

構造・思想・文脈の3つの柱が揃ってはじめて、意味のある作品が生まれる。
